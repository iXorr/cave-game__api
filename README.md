## DESCRIPTION

It's a small browser-based arcade game where the player interacts with a dynamic field in real time. Diamonds and radioactive barrels fall from the top of the screen, and the player’s objective is to collect as many diamonds as possible while avoiding dangerous objects.

The application does not use a database system such as MySQL. Instead, all game-related data is stored directly in separate local folder - ``data``. It is an unusual approach, but it keeps the project simple.
